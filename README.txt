==========================================
  画像比較分析ツール
  AI高解像度変換評価システム
==========================================

【概要】
2つの画像を詳細に比較分析し、どちらが高品質かを数値とグラフで評価します。
AI高解像度変換（chaiNNer、Upscaylなど）の品質比較に最適です。

【初回セットアップ】
1. setup.bat をダブルクリック
   → 自動で仮想環境とライブラリをインストール

【起動方法】
1. 起動.bat をダブルクリック
   → GUIが自動で起動します

【使い方】
1. 「参照...」ボタンで比較したい2つの画像を選択
2. 「分析開始」ボタンをクリック
3. 分析完了後、「📊 わかりやすい解釈」タブで結果を確認
4. 「比較レポートを表示」でグラフ表示
5. 「結果フォルダを開く」で画像を確認

【分析項目】
✓ SSIM（構造類似性）
✓ PSNR（信号対雑音比）
✓ シャープネス（鮮鋭度）
✓ コントラスト
✓ ノイズレベル
✓ エッジ保持率
✓ アーティファクト検出
✓ 色差分析（RGB/HSV/LAB）
✓ 周波数解析
✓ テクスチャ解析
✓ 局所品質マップ

【出力ファイル】
- comparison_report.png : グラフレポート（おすすめ！）
- detailed_analysis.png : 12枚の分析画像
- analysis_results.json : 数値データ（JSON）
- difference.png : 差分画像
- heatmap.png : ヒートマップ

【対応画像形式】
PNG、JPG、JPEG、BMP、TIFF、WebP
※透明背景（RGBA）にも対応

【必要環境】
- Windows 10/11
- Python 3.7以上

【トラブルシューティング】
Q. 起動しない
A. setup.bat を再実行してください

Q. 日本語が文字化けする
A. フォントが自動設定されます（MS Gothic/Yu Gothic/Meiryo）

Q. 分析が遅い
A. 大きな画像（6000px以上）は時間がかかります（数分）

【ファイル構成】
起動.bat              ← これをダブルクリックで起動
setup.bat             ← 初回セットアップ用
image_analyzer_gui.py ← GUIプログラム
advanced_image_analyzer.py ← 分析エンジン
result_interpreter.py ← 結果解釈モジュール
requirements.txt      ← 必要ライブラリ一覧
